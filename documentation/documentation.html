Background Images

Here is where you list all your background images.

js/main.js 
          

$(".bg-images").vegas({
    slides: [
      { src: "img/background1.jpg" },  ('jump',0)
      { src: "img/background2.jpg" },  ('jump',1)
      { src: "img/background3.jpg" },  ('jump',2)
      { src: "img/background4.jpg" }   ('jump',3)
    ],
    delay:300 //Delay between slides in milliseconds.   
});

To fadeIn a new image when hover,add the morph class (in this case .mp_1) and choose the background image (in this case ('jump',1)).

js/main.js 
          
var mp_1 = $(".mp_1").find(morph_title); 
mp_1.on("mouseenter", function(){
  if (!morph.hasClass("open")) {
    main.vegas('options', 'transition', 'fade').vegas('jump',1);
  }
});

To fadeIn a new image when filter,add the button class (in this case .button-a) and choose the background image (in this case ('jump',0)).

js/main.js 
          
var about = $(".button-a"); 
  about.on("click", function(){
  if (!morph.hasClass("open")) {
    main.vegas('options', 'transition', 'fade').vegas('jump',0);
  }
});

Instead of fade you can use other transitions, such as slideRight.

js/main.js 
          
main.vegas('options', 'transition', 'slideRight').vegas('jump',0);

Recommendation: Try not to use really big and heavy images,here's a image compressor.

Contact form


Replace youremail with your actual email to receive messages from the contact form.

js/mailer.php 
          
<?php
if(empty($_POST['name'])   ||   
    empty($_POST['email'])  ||
    empty($_POST['message'])||  
    !filter_var($_POST['email'],FILTER_VALIDATE_EMAIL))   
  {    
       echo "No arguments Provided!";   return false;   
  }
     
  $name = $_POST['name'];
  $email_address = $_POST['email'];
  $message = $_POST['message'];     

 $to = 'youremail'; //HERE YOUR EMAIL

 $email_subject = "Contact form submitted by:  $name"; 
 $email_body = "You have received a new message. \n\n".                
                   " Here are the details:\n \nName: $name \n ".                 
                   "Email: $email_address\n Message \n $message";
 $headers = "From: youremail\n";
 $headers .= "Reply-To: $email_address";    
 
 mail($to,$email_subject,$email_body,$headers); return true;            
?>

Remember : Contact form doesn't work locally.
